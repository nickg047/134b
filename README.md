1.    How to Run

When you first come to our page it will place you on the welcome.html page. From here it request you to create an account with us. You can either create an account yourself or use one of the new login we have provided.
	e-mail: scrumbagstest@gmail.com
	password: ucsd134b
After logging in you can perform the expected functionalities of this app.

  a. Desktop
The following is a link to our released application.
http://acsweb.ucsd.edu/~nlg/cs134b/src/welcome.html

  b. Mobile
The mobile implementation was done through Phone Gap (the free 30 day trial). In 
order to see the app on your phone perform the following steps.
Install the Phone Gap desktop application here: <http://docs.phonegap.com/getting-started/1-install-phonegap/desktop/>
Using your phone install the free Phone Gap application. 
On the desktop version, click the “+” button and choose “Open Existing Project”
Select the folder “Mobile Support” within the main folder
This should create the item/server for the application. It will be notify if the server is running or not. If it is not hit the “>” arrow button on the item to start it.
Copy down IP address given by the desktop application and input it onto your phone and hit connect
Enter login ID if necessary
